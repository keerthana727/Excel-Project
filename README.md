# Bike Sales Dashboard (Excel)

This project presents an interactive Excel dashboard analyzing bike sales data.

## Tools Used
- Microsoft Excel
- Pivot Tables & Charts
- Slicers
- Dashboard Design

## Key Insights
- Income comparison by gender and purchase behavior
- Bike purchases across different age brackets
- Impact of commute distance on purchasing decisions
- Interactive filters for marital status, region, and education
